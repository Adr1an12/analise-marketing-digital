# Regressão Linear - Marketing Digital

Projeto desenvolvido em Python utilizando Machine Learning para prever o número de cliques em campanhas de marketing digital com base na quantidade de anúncios exibidos.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn

## Objetivo

Aplicar Regressão Linear para identificar padrões e realizar previsões utilizando dados históricos de campanhas.

## Funcionalidades

- Leitura de arquivos CSV
- Análise exploratória de dados
- Criação de modelo preditivo
- Visualização gráfica
- Avaliação do modelo
